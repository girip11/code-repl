# Loops in javascript

## for loop
```javascript
// syntax
for(initialization; condition; increment) {
  // statements
}
```

`break` keyword - exit from the loop

`continue` keyword - skip a loop iteration.

## While amd do while loop
```javascript
initialization
while(exit_condition) {
  // statements
  increment
}

initialization
do {
  // statements
  increment
} while(exit-condition);
```
